Intro to GA4

This page provides an introduction to Google Analytics 4 (GA4) for HHS Analytics team members. Learn how to access, navigate, and analyze GA4 data.

What is GA4?

Google Analytics 4 (GA4) is the latest version of Google Analytics, providing website and app analytics. GA4 uses an event-based data model and provides enhanced insights and reporting capabilities.

Key Features

  • Event-Based Tracking: Tracks user interactions as events
  • Enhanced Measurement: Automatic tracking of common events
  • Audiences: Create and analyze user segments
  • Explorations: Custom analysis and reporting
  • API Access: Programmatic data access for analytics

Getting Started

  1. Access GA4 through Google Analytics website
  2. Connect to HHS property
  3. Navigate to Reports section
  4. Explore available metrics and dimensions
  5. Understand key reports and dashboards

Key Metrics

  • Users: Number of unique visitors
  • Sessions: User sessions on the site
  • Page Views: Total page views
  • Engagement Rate: Percentage of engaged sessions
  • Conversions: Goal completions

Access Requirements

  • HHS Google Analytics account access
  • VPN connection (see Section 9-5)
  • Appropriate permissions for property
