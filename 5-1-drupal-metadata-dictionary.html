Drupal Metadata Dictionary

This page documents all metadata fields and attributes available in the Drupal content management system used by HHS. Understanding these fields is essential for content analytics and metadata-driven reporting.

Core Metadata Fields

  • Title: Page title / headline
  • Description: Meta description for SEO
  • Keywords: Meta keywords and tags
  • URL Alias: URL path and slug
  • Published Date: Content publication timestamp
  • Last Modified: Content last updated timestamp

Content Taxonomy Fields

  • Content Type: Article, page, blog post, etc.
  • Category: Content categorization
  • Topic: Subject matter taxonomy
  • Audience: Target audience classification
  • Tags: Additional keywords and labels

Custom Fields

  • Author: Content creator and editor
  • Department: Owning department or team
  • Status: Published, draft, archived, etc.
  • Custom Attributes: Department-specific metadata

Usage in Analytics

This metadata is used in Power BI for:

  • Content inventory analysis
  • Content performance reporting
  • Taxonomy and tagging quality assessment
  • Metadata completeness tracking
