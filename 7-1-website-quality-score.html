Website Quality Score (Baseline Metrics)

This page documents the Website Quality Score methodology and baseline metrics for evaluating website health and performance across HHS.

Quality Score Components

The Website Quality Score is calculated from:

  • Technical SEO: Meta tags, URLs, redirects, page speed
  • Content Quality: Readability, completeness, metadata
  • Accessibility: WCAG compliance, alt text, navigation
  • Performance: Page load time, core web vitals
  • Structure: URL structure, internal linking, navigation

Baseline Metrics

  • Percentage of pages with title tags
  • Percentage of pages with meta descriptions
  • Average page load time
  • Number of broken links
  • Content readability scores

Scoring Methodology

Each component is weighted and scored on a 0-100 scale:

  • Technical SEO: 30%
  • Content Quality: 25%
  • Accessibility: 20%
  • Performance: 15%
  • Structure: 10%

Tracking & Improvement

Quality scores are tracked over time to:

  • Identify areas needing improvement
  • Measure impact of optimization efforts
  • Compare performance across content areas
  • Set quality improvement goals
