Intro to Sprinklr Analytics

This page provides an introduction to Sprinklr analytics for HHS Analytics team members. Learn how to access and analyze social media data using Sprinklr.

What is Sprinklr?

Sprinklr is a unified customer experience management platform that provides social media management, analytics, and engagement tracking across multiple platforms.

Key Features

  • Social Media Management: Manage posts across multiple platforms
  • Analytics & Reporting: Track performance and engagement
  • Engagement Tracking: Monitor likes, comments, shares, and followers
  • Campaign Management: Track and analyze campaign performance
  • API Access: Programmatic data access for analytics

Getting Started

  1. Access Sprinklr through web interface
  2. Navigate to Analytics section
  3. Explore available reports and dashboards
  4. Understand key metrics and dimensions
  5. Learn API access for Power BI integration

Key Metrics

  • Followers: Follower counts and growth
  • Engagement: Likes, comments, shares
  • Reach: Number of users who saw content
  • Impressions: Number of times content was displayed
  • Engagement Rate: Percentage of engaged users
