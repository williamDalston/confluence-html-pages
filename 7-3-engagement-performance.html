Engagement & Page Performance Metrics

This page documents engagement and page performance metrics tracked in HHS KPI dashboards, including user behavior, content performance, and website analytics.

Engagement Metrics

  • Page Views: Total number of page views
  • Unique Visitors: Number of distinct users
  • Bounce Rate: Percentage of single-page sessions
  • Average Session Duration: Average time spent on site
  • Pages per Session: Average pages viewed per visit

Content Performance

  • Top Pages: Most viewed content pages
  • Entry Pages: Most common landing pages
  • Exit Pages: Pages where users leave
  • Scroll Depth: How far users scroll on pages
  • Time on Page: Average time spent on individual pages

Page Performance

  • Page Load Time: Time to load page content
  • Core Web Vitals: LCP, FID, CLS scores
  • Error Rate: Percentage of pages with errors
  • Mobile Performance: Mobile-specific metrics

Data Sources

Metrics are sourced from:

  • Google Analytics 4 (GA4) data
  • Website crawl data from Screaming Frog
  • Page performance monitoring tools

Dashboard Visualizations

These metrics are visualized in:

  • HHS KPI Dashboard (Section 7)
  • Content Performance Dashboard
  • Website Quality Score (Section 7-1)
