Custom Tag Extraction Examples (XPath examples)

This page contains XPath examples for extracting custom tags and metadata from website content using Screaming Frog or other crawling tools.

XPath Extraction Patterns

Metadata Tag Extraction

Extract custom metadata attributes

//meta[@name='custom-field']/@content
//article/@data-topic
//div[@class='content-tags']/span/text()

Taxonomy Extraction

Extract taxonomy terms and categories

//div[@class='topic-tags']//a/text()
//span[@class='category']/text()
//ul[@class='taxonomy-list']/li/a/text()

Content Type Detection

Identify content type from page structure

//body/@data-content-type
//article/@itemtype
//div[@class='content-wrapper']/@data-type

Custom Attributes

Examples of custom attributes that can be extracted:

  • Department or owner
  • Publication date and author
  • Content classification
  • Campaign or initiative tags

Integration

Extracted tags can be used in:

  • Content inventory analysis
  • Tagging quality assessment
  • Taxonomy gap analysis
  • Power BI content models
