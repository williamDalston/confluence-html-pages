Example API Calls (HTML widget)

This page contains example Sprinklr API calls and request/response patterns. Use these examples as templates when integrating Sprinklr data into Power BI or other analytics tools.

API Endpoints

Follower Analytics

Endpoint for retrieving follower counts and growth metrics

GET /v2/analytics/followers
Query Parameters:
  - platform: twitter, facebook, instagram
  - date_range: start_date, end_date
  - metrics: follower_count, growth_rate

Engagement Metrics

Endpoint for retrieving engagement data

GET /v2/analytics/engagement
Query Parameters:
  - content_type: posts, comments, shares
  - time_period: daily, weekly, monthly

Request Headers

  • Authorization: Bearer {API_KEY}
  • Content-Type: application/json

Response Format

API responses are in JSON format and include:

  • Metrics data (counts, rates, percentages)
  • Date/time information
  • Platform and content type identifiers
