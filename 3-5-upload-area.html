Upload Area (CSV / Excel from Crawls)

This page serves as a repository for Screaming Frog crawl exports. Upload CSV or Excel files from website crawls for storage, sharing, and integration with Power BI.

Upload Guidelines

When uploading crawl data:

  • File Naming: Use descriptive names with date, e.g., "full-crawl-2024-01.csv"
  • File Format: CSV or Excel (.xlsx) format preferred
  • Metadata: Include a description of crawl type and configuration
  • Version Control: Update existing files rather than creating duplicates

File Organization

  • Full Site Crawls: Comprehensive crawls covering entire website
  • Custom Extractions: Focused crawls with specific data extraction
  • Historical Archives: Older crawls maintained for trend analysis

Integration with Power BI

Uploaded crawl data can be:

  • Imported directly into Power BI datasets
  • Combined with other data sources for comprehensive analysis
  • Used to create content inventory and SEO dashboards
  • Compared with historical data for trend analysis

Access & Permissions

Note: Upload area is accessible to analytics team members. Ensure proper file naming and organization to maintain clarity and ease of access.
