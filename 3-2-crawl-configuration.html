Crawl Configuration Settings

This page documents the recommended crawl configuration settings for Screaming Frog crawls used in HHS Analytics. Proper configuration ensures accurate and complete data collection.

Basic Configuration

Mode Settings

  • Crawl Mode: Spider (for HTML crawling) or List (for specific URL lists)
  • Protocol: HTTP/HTTPS - match your site protocol
  • Store HTML: Enable for content analysis

Crawl Limits

  • Max URLs: Set appropriate limit based on site size
  • Max Depth: Control crawl depth to focus on important pages
  • Max File Size: Limit large file downloads to save time

Advanced Settings

Respect & Follow

  • robots.txt: Respect robots directives
  • Canonical Tags: Follow canonical URLs
  • Noindex: Choose whether to crawl noindex pages

Authentication

  • Basic Auth: Configure if site requires authentication
  • VPN Requirements: Ensure VPN is connected if required (see Section 9)

Custom Extraction Settings

Configure custom extraction rules based on your analysis needs:

  • XPath selectors for specific elements
  • CSS selectors for content extraction
  • Regex patterns for pattern matching

Recommended Configurations by Use Case

  • Full Site Audit: Comprehensive crawl with all resources
  • Quick Health Check: Fast crawl focusing on critical issues
  • Content Extraction: Configured for specific data extraction
