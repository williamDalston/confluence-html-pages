Power BI Embeds (Mock Dashboards)

This page contains embedded Power BI dashboards and mockups. These dashboards demonstrate key metrics and visualizations for various analytics use cases.

Available Dashboards

  • HHS KPI Dashboard: Executive-level overview of key performance indicators
  • Website Quality Score: Baseline metrics for website performance and quality
  • Team SLA Dashboard: Service desk performance and SLA compliance tracking
  • Social Trends Dashboard: Sprinklr analytics and engagement metrics
  • Content Performance Dashboard: Content analytics and metadata insights

Embedding Instructions

To embed Power BI dashboards in Confluence:

  1. Publish your Power BI report to Power BI Service
  2. Copy the embed URL or iframe code
  3. In Confluence, insert an HTML widget
  4. Paste the iframe embed code
  5. Adjust sizing parameters as needed
