Intro to Screaming Frog

This page provides an introduction to Screaming Frog SEO Spider for HHS Analytics team members. Learn how to use Screaming Frog for website crawling and analysis.

What is Screaming Frog?

Screaming Frog SEO Spider is a website crawling tool that crawls websites' links, images, CSS, scripts, and apps from an SEO perspective, analyzing key elements and generating reports.

Key Features

  • Website Crawling: Discover all pages, links, and assets
  • Technical SEO Audits: Identify SEO issues and opportunities
  • Custom Extraction: Extract specific data using XPath or CSS selectors
  • Data Export: Export crawl data to CSV, Excel, or XML
  • Scheduling: Automate crawls on a schedule

Getting Started

  1. Install Screaming Frog SEO Spider
  2. Configure crawl settings (see Section 3-2)
  3. Enter website URL to crawl
  4. Start crawl and review results
  5. Export data for analysis

Common Use Cases

  • Technical SEO audits
  • Content inventory and analysis
  • Broken link detection
  • Metadata extraction
  • Custom data extraction for Power BI
