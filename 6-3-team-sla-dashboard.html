Team SLA Dashboard Mockup

This page contains mockups and specifications for the Team SLA Dashboard, which tracks service desk performance and SLA compliance metrics across teams.

Dashboard Overview

The Team SLA Dashboard provides:

  • Real-time SLA compliance tracking
  • Team performance comparisons
  • Ticket volume and trend analysis
  • Resolution time metrics

Key Visualizations

  • SLA Compliance Rate: Gauge or percentage display
  • Ticket Volume by Team: Bar chart or table
  • Average Resolution Time: Line chart over time
  • Status Distribution: Pie or donut chart
  • Team Performance Comparison: Matrix or table

Key Metrics

  • SLA compliance percentage
  • Average resolution time
  • Ticket volume and trends
  • Status transition rates
  • Backlog size and age

Data Sources

Dashboard is powered by:

  • Fact_TicketHistory model (Section 6-2)
  • Jira API data (Section 6-1)
  • Real-time or daily refresh depending on use case
