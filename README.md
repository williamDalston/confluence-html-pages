# Confluence HTML Pages for Power BI

This repository contains HTML content designed for embedding into Confluence pages via HTML widgets. These pages support the HHS Analytics Power BI workspace documentation.

## Usage

1. Open the desired HTML file (e.g., `page1-purpose.html`)
2. Copy the entire content inside the file
3. In Confluence, insert an HTML widget
4. Paste the copied HTML content into the widget
5. Save your Confluence page

## Pages

- **page1-purpose.html** - Purpose of This Power BI Workspace (HHS Analytics)
  - Central knowledge hub overview
  - What the page provides
  - Target audience
  - Why it matters

## Structure

Each HTML file contains self-contained content wrapped in a div suitable for Confluence widget embedding. The styling is inline to ensure compatibility with Confluence's HTML widget restrictions.
