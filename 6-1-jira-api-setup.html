Jira API Setup & Queries

This page provides setup instructions and example queries for connecting to the Jira REST API for ticket tracking and team workflow analytics.

API Setup

Authentication

  • Method: Basic authentication with API token
  • Token Generation: Create API token in Jira account settings
  • Permissions: Read access to tickets and projects

Base URL

https://[your-domain].atlassian.net/rest/api/3/

Common Queries

Get Tickets

GET /search
Query Parameters:
  - jql: JQL query string
  - fields: comma-separated field list
  - maxResults: number of results (max 100)

Get Ticket History

GET /issue/{issueId}/changelog

JQL Examples

  • project = "HHS" AND status = "Open"
  • assignee = currentUser() AND updated >= -7d
  • status changed FROM "In Progress" TO "Done" DURING (startDate(), endDate())
