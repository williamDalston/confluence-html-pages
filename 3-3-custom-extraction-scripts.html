Custom Extraction Scripts

This page contains custom extraction scripts and XPath examples for extracting specific data from website crawls using Screaming Frog.

Extraction Methods

  • XPath: For navigating XML/HTML structure and extracting elements
  • CSS Selectors: For selecting elements using CSS syntax
  • Regex: For pattern matching in text content

Common Extraction Patterns

Metadata Extraction

  • Page title, meta description, meta keywords
  • Open Graph tags and Twitter Card metadata
  • Schema.org structured data

Content Extraction

  • H1, H2, H3 headings hierarchy
  • Body content and text length
  • Image alt text and captions

Custom Data Points

  • Publication dates and author information
  • Category tags and taxonomy
  • Custom metadata fields from CMS

XPath Examples

Note: Full XPath examples are stored in Confluence code widgets. Navigate to specific sections to view complete extraction scripts.

Common XPath patterns:

  • //meta[@name='description']/@content - Extract meta description
  • //h1/text() - Extract H1 heading text
  • //article/@data-published-date - Extract custom data attributes
