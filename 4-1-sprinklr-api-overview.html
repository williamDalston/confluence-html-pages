Sprinklr API Overview

This page provides an overview of the Sprinklr API used for accessing social media analytics, engagement data, and follower tracking in HHS Analytics.

What is Sprinklr?

Sprinklr is a unified customer experience management platform that provides social media management, analytics, and engagement tracking across multiple platforms.

API Capabilities

  • Social Media Data: Access to posts, comments, likes, shares, and engagement metrics
  • Follower Analytics: Track follower counts, growth trends, and audience demographics
  • Content Performance: Analyze content reach, impressions, and engagement rates
  • Campaign Tracking: Monitor campaign performance and ROI metrics

Authentication

  • Method: API key authentication
  • Access Levels: Read-only access for analytics data
  • Token Management: See Section 9 for API token governance

Integration with Power BI

Sprinklr API data is integrated into Power BI for:

  • Social trends dashboards
  • Engagement rate analysis
  • Follower growth tracking
  • Campaign performance monitoring
