Connection Guides

This page provides step-by-step guides for connecting Power BI to various data sources used in HHS Analytics, including Jira, GA4, and Sprinklr.

Available Connection Guides

Jira Connection

Connect Power BI to Jira using the Jira REST API or Jira Cloud connector.

  • Authentication method: API token
  • Required permissions: Read access to tickets and projects
  • Data refresh frequency: Daily or hourly depending on use case
  • See Section 6 for detailed Jira API setup and queries

Google Analytics 4 (GA4) Connection

Connect Power BI to GA4 using the GA4 connector or API integration.

  • Authentication method: OAuth 2.0
  • Required permissions: Read access to analytics data
  • Data refresh frequency: Daily recommended
  • Note: Be aware of GA4 token limits (see Section 8)

Sprinklr Connection

Connect Power BI to Sprinklr using the Sprinklr API.

  • Authentication method: API key
  • Required permissions: Read access to analytics and engagement data
  • Data refresh frequency: Daily or as needed
  • See Section 4 for detailed Sprinklr API overview and examples

Connection Best Practices

  • Always use parameterized queries for flexibility
  • Implement incremental refresh where possible
  • Store credentials securely in Power BI Service
  • Document any custom M scripts or transformations
  • Test connections before deploying to production
