Available Metrics & Objects

This page documents all available metrics and data objects accessible through the Sprinklr API for HHS Analytics use cases.

Engagement Metrics

  • Likes: Number of likes/reactions on posts
  • Comments: Number of comments on posts
  • Shares: Number of shares/reposts
  • Impressions: Number of times content was displayed
  • Reach: Number of unique users who saw content
  • Engagement Rate: Percentage of engaged users

Follower Metrics

  • Follower Count: Total number of followers
  • Follower Growth: Net change in followers over time
  • Follower Demographics: Age, gender, location breakdowns

Content Metrics

  • Post Count: Number of posts published
  • Content Type: Text, image, video, link
  • Publish Date: When content was published
  • Platform: Twitter, Facebook, Instagram, LinkedIn, etc.

Campaign Metrics

  • Campaign Name: Identifier for campaign grouping
  • Campaign Performance: Aggregated metrics per campaign
  • ROI Metrics: Return on investment calculations

Data Objects

Available data objects include:

  • Posts, Comments, Messages
  • Users, Followers, Audiences
  • Campaigns, Ads, Content
  • Platforms, Accounts, Pages
