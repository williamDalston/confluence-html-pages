Rows.com Embedded Charts (Follower Tracking)

This page contains embedded charts from Rows.com for tracking social media follower growth and engagement trends over time.

What is Rows.com?

Rows.com is a collaborative spreadsheet tool that enables data visualization and chart embedding. We use it for tracking social media metrics and follower trends.

Available Charts

  • Follower Growth Chart: Track follower count changes over time
  • Platform Comparison: Compare follower metrics across platforms
  • Engagement Trends: Monitor engagement rate changes
  • Content Performance: Visualize top-performing content

Data Sources

Charts are populated from:

  • Sprinklr API data
  • Historical follower tracking data
  • Engagement metrics from social platforms

Embedding Instructions

To embed Rows.com charts in Confluence:

  1. Create or open your chart in Rows.com
  2. Get the embed URL or iframe code
  3. In Confluence, insert an HTML widget
  4. Paste the embed code
  5. Adjust sizing as needed

Updates

Charts are updated automatically based on data refresh cadence (daily or weekly). See Section 8 for metrics cadence details.
