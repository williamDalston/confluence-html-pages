Social Trends Dashboard (Sprinklr + Rows)

This page documents the Social Trends Dashboard, which combines Sprinklr analytics and Rows.com embedded charts to track social media performance and engagement.

Dashboard Overview

The Social Trends Dashboard provides:

  • Follower growth trends across platforms
  • Engagement rate analysis
  • Content performance rankings
  • Platform comparison metrics

Key Visualizations

  • Follower Growth Chart: Line chart showing follower count over time
  • Engagement Rate Trend: Line chart showing engagement trends
  • Platform Comparison: Bar chart comparing metrics across platforms
  • Top Performing Content: Table or bar chart of best-performing posts

Data Sources

Dashboard combines data from:

  • Sprinklr API: Social media metrics and engagement data
  • Rows.com: Embedded charts for follower tracking
  • Power BI: Aggregated analysis and trend visualization

Key Metrics

  • Follower count and growth rate
  • Engagement rate (likes, comments, shares)
  • Content reach and impressions
  • Campaign performance metrics

Refresh Cadence

Dashboard is updated:

  • Daily for follower counts and engagement
  • Weekly for content performance summaries
