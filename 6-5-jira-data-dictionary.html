Data Dictionary for Jira Fields

This page provides a comprehensive data dictionary of all Jira fields used in analytics, including definitions, data types, and usage notes.

Core Ticket Fields

  • Issue Key: Unique identifier (e.g., HHS-123)
  • Summary: Ticket title or headline
  • Description: Detailed ticket description
  • Issue Type: Bug, Story, Task, Epic, etc.
  • Status: Current status (Open, In Progress, Done, etc.)
  • Priority: Critical, High, Medium, Low

Assignment Fields

  • Assignee: User currently assigned to ticket
  • Reporter: User who created the ticket
  • Component: System component or area
  • Team: Owning team or department

Date Fields

  • Created: Ticket creation timestamp
  • Updated: Last update timestamp
  • Resolved: Resolution timestamp (if resolved)
  • Due Date: Target completion date

Custom Fields

  • Labels: Free-form tags for categorization
  • Epic Link: Link to parent epic
  • Sprint: Sprint assignment (for Agile teams)
  • Story Points: Effort estimation
